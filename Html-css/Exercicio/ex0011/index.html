Usando um link externo

Devemos usar a tag A(significa Âncora) para ancorar um site no Html.
Exemplo:
Meu canal do youtube.
Podemos usar da seguinte forma também:
Meu personagem favorito é o: https://www.google.com/url?sa=i&url=https%3A%2F%2Fwww.deviantart.com%2Fnoobmanek%2Fart%2FGoku-dando-o-dedo-947169391&psig=AOvVaw2ppOjelmVukIfN2aVFgIO8&ust=1708552087662000&source=images&cd=vfe&opi=89978449&ved=0CBIQjRxqFwoTCIignNLyuoQDFQAAAAAdAAAAABAI

Usando desta forma, apenas com a tag A, o seu site é sobreposto pelo link, e não queremos isso.
Portanto, usaremos a tag A em conjunto com a seguinte tag:
"target="_blank" rel="external" ";
