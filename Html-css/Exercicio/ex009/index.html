A tag code é usada para fazer coisas assim
system.out.printlf("Eu sou o Arthur, tenha um bom dia!");

Quando você usa a tag "pre", tudo que está dentro dela é considerado, inclusive os espaços, parágrafos e etc.
ex: 

            O goku é muito forte
            lero
                        lero
                                lero
                                hahaahahk
                                eu sou o maior fan dele!

Ou seja, tudo que estiver dentro do pre vai aparecer, independentemente do que!
